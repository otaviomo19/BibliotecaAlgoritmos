A distância de edição é o número mínimo de inserções, remoções e substituições
que devem ser realizadas para fazer com que uma cadeia de caracteres
A seja igual a uma segunda cadeia de caracteres B. Uma das maneiras mais
comuns de resolver este problema é através do uso da programação dinâmica.
Assim, os dados são tabelados para evitar o recálculo das várias
combinações possíveis. O algoritmo para calcular a distância de edição utilizando
a técnica de programação dinâmica é O(MxN), onde m corresponde
ao tamanho de A e n ao tamanho de B.

Aplicações : Há muitas aplicações práticas do algoritmo de distância de edição, exibir todas 
as palavras em um dicionário que estão perto de proximidade com uma determinada 
palavra ou palavra escrita incorretamente.


****** Para Executar *****
há 2 arquivos txt, o arquivo "Entrada", são as entradas
para o programa, um par de palavras, tal que o padrão de entrada é "palavra1 palavra2", 
e em seguida ele grava os resultados no arquivo "Resultados".
